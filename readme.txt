1. The install videos folder contains some videos about the installation and use of the printer

2. The Test models folder is a model of some test printers

3. The v400 config for CURA5.0 folder is the configuration file for CURA5.0 to configure the v400 model

4. The v400 config for CURA4.13.1 folder is the configuration file for CURA4.13.1 to configure the v400 model

5. You can install Ultimaker Cura 4.13.1.exe file and Ultimaker Cura 5.0.0.exe file to install cura4.13.1 and cura5.0 and add v400 models directly
